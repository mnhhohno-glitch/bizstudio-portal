## 1. ミドル層 vs 若手層の根本的な違い

| 項目 | 若手層（20代前半〜30代前半） | ミドル層（35歳以上） |
|---|---|---|
| **市場の評価基準** | ポテンシャル・将来性 | 即戦力・実績 |
| **業界チェンジ** | 未経験歓迎求人が豊富 | 経験業界が基本、異業種は極めて困難 |
| **書類通過の鍵** | 年齢と基本スペック | 職務経歴書の構成と打ち出し方 |
| **提案の自由度** | 幅広く提案可能 | 通る求人が限られるため精度重視 |
| **年収交渉** | ベースが低いため上がりやすい | 現年収維持が最大の課題になることが多い |
| **温度感管理** | やる気の維持がメイン | 現実認識と折り合いの支援がメイン |

## 2. 段階的提案戦略の詳細

### フェーズ1：希望でグリップを握る（初回〜2回目面談）

- 本人の希望をそのまま受け止めて求人を提案する
- 「それは厳しいですよ」と最初から言わない
- 希望業界の求人に実際にエントリーしてもらう
- この段階では「信頼関係の構築」が最優先

### フェーズ2：市場価値の自覚（書類選考結果が出た頃）

- 書類で落ちた結果をファクトとして共有する
- 「○件出して○件落ちたので、通過率はX%です」と数字で示す
- 感情的にならず、データとして現実を提示する
- ここで初めて「ちょっと別の角度でも探しませんか」と切り出す

### フェーズ3：現実的な提案への移行

- 経験業界・経験職種でより良い環境の求人を提案
- 「業界は同じですが、この会社は商材が違いますし将来性があります」
- 同業界の成長企業・DX推進企業など、本人の不満を解決できる求人
- 異業種でもマネジメント経験を武器にできるポジションを提案

## 3. 業界チェンジの現実ライン

### 35歳以上で業界チェンジが通る条件

- **マネジメント経験が5年以上**：業界よりもマネジメント能力で採用される
- **業界を跨ぐ汎用スキル**：英語力、DX推進経験、新規事業立ち上げ等
- **受け入れ企業の事情**：急成長中で業界経験よりマネジメント人材が欲しい企業
- **中堅企業・ベンチャー**：大手よりも実力主義で学歴・業界経歴にこだわらない

### 35歳以上で業界チェンジがほぼ不可能なケース

- B2C経験のみでB2B希望（営業スタイルが根本的に異なる）
- 特殊業界（教育・公共・自衛隊等）のみの経験で一般企業の営業希望
- 高卒+特殊業界のみの経歴で大手メーカー・商社希望

### 対応策

上記の「ほぼ不可能」ケースでも、最初から否定しない。フェーズ1→2→3の段階的アプローチで本人に自覚させてから、現実的な提案に移行する。

## 4. 職務経歴書のポジショニング変換

ミドル層の書類通過は職務経歴書の打ち出し方が最大のレバー。

### 変換の例

| 元の記載 | 変換後 |
|---|---|
| 「英会話スクールのエリアマネージャー」 | 「50名規模の営業組織マネジメント（P/L管理、採用、育成）」 |
| 「教育委員会向けデジタルコンテンツ提案」 | 「自治体向けB2B営業（DX推進・新規開拓）」 |
| 「百科事典の販売」 | 「教育系SaaS/デジタルコンテンツの法人営業」 |

### ポイント

- **業界固有の用語を汎用的な表現に置き換える**
- **数字を入れる**：売上金額、達成率、マネジメント人数、顧客数
- **「何をしたか」ではなく「どんな成果を出したか」にフォーカス**
- **推薦状より職務経歴書の方が書類通過への影響が大きい**
