## 4. 新ヘッダ仕様

### Row 1（上段）
- 左: 氏名（19px太字）+ フリガナ・ID・登録日・担当（13px）
- 右: 状態select / 中項目 / 基本情報編集ボタン（各130px × 32px）
- 氏名・IDはクリックコピー対応

### Row 2（中段）
- 生年月日(年齢) → 性別 → 電話 → メール → 住所
- 全項目クリックコピー対応（sonner トースト表示）
- ホバーでコピーアイコン表示

### Row 3（下段）
- URL・資料ボタン群: 求人マイページ / ガイドURL / 日程調整URL / 求人出力

- 直近5件を横並びボタンで表示
- 6件目以降は「すべて(N)」ドロップダウン
- 状態アイコン: ●緑(complete) / ●黄(draft+保存あり) / ●赤(draft+未保存)

- 右端: PDF表示 / キャンセル / 保存ボタン
